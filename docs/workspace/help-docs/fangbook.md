## FangCore

一生一芯第四期项目中实现的处理器核

- 支持RISCV64 IM指令
- 经典五级流水
- Booth移位乘法器、移位除法器
- 实现AXI4总线规范
- 4KB ICache、4KB Dcache
- 实现M模式，支持环境调用异常、时钟中断
- 可运行RT-thread
